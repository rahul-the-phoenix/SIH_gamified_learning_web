# UDAAN - Student & Teacher Portal!!!

## 🌟 Project Overview

**UDAAN** (Wings to Learn) is an interactive educational web application designed to make learning engaging through gamification, interactive animations, and quizzes. The platform provides a comprehensive learning experience with a focus on physics concepts, particularly gravity and planetary science.

## ✨ Features

### 1. **User Authentication**
- Student and Teacher role selection
- Sign-up form with validation (email/phone & password)
- Secure user registration flow

### 2. **Student Dashboard**
- **Student Details**: Personal information, level, XP points
- **Achievements**: Track badges, streaks, and accomplishments
- **Progress Tracker**: Visual representation of completed vs pending tasks with interactive charts
- **Work Done**: List of completed learning modules
- **Logout Functionality**: Secure session management

### 3. **Gravity Simulator**
- **Interactive Animation**: Watch balls fall on different planets with varying gravitational forces
- **Multi-Planet Support**: 
  - 🌍 Earth (9.8 m/s²)
  - 🌝 Moon (1.62 m/s²)
  - 🌕 Jupiter (24.79 m/s²)
  - ☀ Sun (274 m/s²)
  - 🌑 Venus (8.87 m/s²)
- **Controls**: Start/Pause and Reset animations
- **Quick Quiz**: Integrated quiz about planetary gravity

### 4. **Quiz System**
- **Gravity Knowledge Quiz**: 5 comprehensive questions
- **Instant Results**: Score calculation with detailed answer review
- **Try Again Option**: Reattempt quizzes for better learning
- **Question Types**: Multiple-choice questions covering:
  - Strongest gravitational force
  - Weight comparison across planets
  - Einstein's theory of relativity
  - Mars gravity comparison
  - Factors determining planet gravity

### 5. **Visual Features**
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Gradient backgrounds, smooth animations, and hover effects
- **Interactive Charts**: Progress tracking with doughnut charts (Chart.js)
- **Loading States**: Visual feedback during form submission
- **Gradient Buttons**: Attractive and intuitive UI elements

## 🚀 How to Use

1. **Start**: Select "Student" or "Teacher" role
2. **Sign Up**: Enter your details (name, email/phone, password)
3. **Choose Activity**:
   - View Student Dashboard for progress tracking
   - Explore Gravity Simulator
   - Take the Gravity Quiz
4. **Navigate**: Use back buttons to return to previous pages
5. **Logout**: Use the logout option in the dashboard
